### 实现的功能
1.音乐播放、暂停、上一曲、下一曲
2.切换播放模式，顺序播放、单曲循环、随机播放
3.热门歌单、歌手列表、排行榜
4.歌词显示
5.热门搜索、搜索歌手、歌曲
6.记录搜索历史，播放历史
7.收藏歌曲
